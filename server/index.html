PDF Generator

Getting Started

Basically, get yourself a plugin or something that allows you to make post requests. I personally use Postman or curl. If you are seeing this, then you have already started the server with "npm start".

With the server running, make a post request to the server to any of the post endpoints. Look in index.js for the "app.post" blocks to know the valid ones.

To send data to the templates, look in the template you want to render and look for the replacement variables. Just add a JSON to the string to the request and add that variable to the json. For example:

{ foo: "My foo value" }

This will replace {{ foo }} with "My foo value".

Example Call for Test

curl -H "Content-Type: application/json" -X POST -d '{ "title": "My PDF title" }' http://localhost:3000/test > ~/Test.pdf; open ~/Test.pdf

Change the title by editing the above command and replacing "My PDF title" with anything you want.
